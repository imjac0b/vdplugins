# Discord Mobile Mod Plugins

A collection of plugins compatible with Discord mobile client modifications:

- [Vendetta](https://github.com/vendetta-mod/Vendetta)
- [Bunny](https://github.com/bunny-mod/Bunny)
- [Revenge](https://github.com/revenge-mod/revenge-bundle)
- [Kettu](https://codeberg.org/cocobo1/Kettu)

## Plugins

### CustomPFP

Customize your Discord profile with custom profile pictures (static & animated), banners, and Discord badges.

**Features:**

- Custom static and animated profile pictures
- Custom profile banners
- Add Discord badges (Staff, Partner, HypeSquad, Bug Hunter, Early Supporter, Verified Developer, Certified Moderator, Active Developer, and more)

**Installation:**
Paste the following URL into the Plugins page of your Discord mod:

```
https://imjac0b.github.io/vdplugins/custompfp
```

## How to install plugins?

Paste a plugin URL into the Plugins page of your Discord mod (Vendetta/Bunny/Revenge/Kettu), following the format:

```
https://imjac0b.github.io/vdplugins/PLUGIN_NAME
```
